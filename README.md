# RevuLink
Bringing Social media User voice to E-commerce

RevuLink is a ecommerce platform plugin that allows users to get the feedback of a particular product from across the social media and the ai gives the overall rating . 

The advantage is that usually customers need to rely on comments from the ecommerce platform this add them an option to search for user video buyer suggesion video from youtube 

The repo contains the basic version 1.0 of the plugin
